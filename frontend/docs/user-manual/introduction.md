# Introduction

**ENIC MIS** (Management Information System) is an internal web application built to support the processing and validation of Philippine government hospital claims data.

This manual is for **claims processors and operations staff** who use the system daily. No technical background is required.

## What You Can Do

- **Upload** T3 and T4 spreadsheet files (`.xlsx`, `.csv`, `.tsv`, `.xls`)
- **Inspect and clean** claims data — filter, search, sort, and correct records
- **Select rows** individually, by page, or all filtered results at once
- **Export** selected rows in multiple formats — Excel, CSV, TSV, or XML
- **Manage users** and control role-based access (admin only)

## Roles in the System

| Role | Access |
|---|---|
| `guest` | Limited read access |
| `operations` | Data Cleaning Tool |
| `managers` | Data Cleaning + Reports |
| `admin` | Full access except superadmin functions |
| `superadmin` | Full system access |
| `dev` | Developer access |
| `it` | IT management |
| `marketing` | Marketing pages |
| `documentations` | Documentation access |

## Before You Start

Make sure you have:
- A registered account (contact your supervisor if you don't have one)
- Your T3/T4 spreadsheet file ready to upload
- The ENIC MIS URL from your team
