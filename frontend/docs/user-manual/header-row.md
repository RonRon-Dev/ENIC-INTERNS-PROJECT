# Setting the Header Row

After uploading, the **Select Header Row** dialog appears showing a preview of the first rows of your file.

## Why This Step Exists

Some T3/T4 files have extra title rows or blank rows above the actual column headers. This step lets you tell the system exactly which row contains your column names.

## How to Select

1. The dialog shows the first several rows of the file as a table
2. Click the row that contains your **column names** (e.g. `ClaimNo`, `MemberName`, `Amount`)
3. That row will be highlighted
4. Click **Use Row** to confirm

::: tip
Look for the row with recognizable column names like `ClaimNo`, `HospitalCode`, `ChequeDate`, `VoucherDate`, etc.
:::

## After Confirming

The **Configure Columns** dialog opens automatically next.
