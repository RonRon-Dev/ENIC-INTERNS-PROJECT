# Selecting Rows

You must select rows before you can export them.

## How to Select

| Action | How |
|---|---|
| Select a single row | Click the checkbox on the left of the row |
| Select all on current page | Click the checkbox in the table header |
| Select all filtered results | After selecting the page, click **"Select all X rows"** banner that appears |
| Deselect all | Click the header checkbox again, or click **Clear (N)** in the toolbar |

## Selection Count

The toolbar always shows how many rows are currently selected via the **Export** button label — e.g. `Export 142`.

## Clear Button

When rows are selected, a **Clear (N)** button appears in the toolbar. Clicking it deselects all rows.

::: tip
Selection is maintained across pages — selecting rows on page 1 then navigating to page 2 keeps page 1 rows selected.
:::

## After Selecting

Once at least one row is selected, the **Export** button becomes active.
